Python web api project.

Goal: build a command-line python app to fetch wether 
and crypto prices using web APIs. 

Setup:
We will make a project folder named python-api-project

we intialized git on the project.
git init

we created virtual env and activated it.
python -m venv venv
venv\Scripts\activate

we installed requests and python-dotenv.
command - pip install requests python-dotenv

we created a .gitignore file and filled it with

venv/
.env
__pycache__/


WE created the .env.example file

WEATHER_API_KEY=your_openweathermap_api_key
